# Calculator App

A beautiful, responsive calculator web application built with HTML, CSS, and JavaScript.

![Calculator Preview](https://img.shields.io/badge/status-active-success) ![License](https://img.shields.io/badge/license-MIT-blue)

- **Basic Operations**: Addition, subtraction, multiplication, division
- **Modern UI**: Glassmorphism design with smooth animations
- **Keyboard Support**: Full keyboard compatibility
- **Responsive Design**: Works on all devices (mobile, tablet, desktop)
- **Error Handling**: Prevents invalid operations
- **Decimal Operations**: Supports floating-point calculations
